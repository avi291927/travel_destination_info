📖 About

WanderWorld is a travel destination information platform built with HTML, CSS, and JavaScript that allows users to browse popular travel spots, explore destination categories, view featured places, and get inspired for trips around the world.  ￼

This project can be used as a frontend showcase for travel guides, tourism blogs, or destination discovery dashboards.

⸻

✨ Features

✔ Home page with scrollable featured content
✔ Explore destinations by category (Beach, Mountain, City, Cultural, Adventure, Nature)
✔ User interface for destination browsing
✔ Responsive layout for mobile and desktop devices
✔ Navigation menu with links to login/signup/dashboard pages (UI only)  ￼

⸻
